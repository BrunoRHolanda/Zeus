Cadastrar conta

UMLUseCase
Zeus-uml :: DIM. de casos de uso :: Gerenciar conta no banco :: Cadastrar conta

Description

Para o cadastro das contas são imprescindíveis o tipo, número, cliente e um saldo, que para novas contas não deve ser inferior a ser 150 reais. As contas podem ser conta corrente ou conta poupança que será escolhido pelo cliente no ato da abertura.
